Kidzplay

Verslag

In de cursus die ik gevolgd heb wordt er aan de hand van de Mern-stack een e-commerce platform gebouwd. Ik ben bij deze cursus terecht gekomen omdat ik React.js wilde leren maar ik niet enkel puur React maar in combinatie met andere technologieën. Bovendien kwam mijn vriendin met het idee kwam om een website/platform te creëren waar mensen speelgoed kunnen huren. Bij het zoeken kwam ik al snel bij de MERN stack terecht. De MERN-stack is een combinatie van de technologieën MongoDb-Express.js-react.js-node.js. Het is een Javascript stack waarmee op een relatieve snelle manier een full stack applicatie mee ontwikkeld kan worden. In dit verslag zal ik een overzicht geven van de cursus die ik heb gevolgd, waarbij ik de belangrijkste onderwerpen bespreek en toelicht.



De cursus begon met de basisprincipes van React waarbij de nadruk lag op het gebruik van functionele componenten en React hooks. In React werk je ook met componenten, zoals in Angular. Door deze component gebaseerde architectuur is het gemakkelijk om herbruikbare en modulaire UI-elementen te bouwen voor in de frontend. React heeft ook een React bootstrap library die geïmporteerd kan worden en het mogelijk maakt om elementen te gebruiken die het creëren en stylen van componenten vergemakkelijkt. Bovendien heb ik ook geleerd hoe met behulp van React Router client-side routering geïmplementeerd kan worden en navigatie tussen verschillende pagina’s afgehandeld kan worden. Aan het gebruik van React Hooks werd ook aandacht besteed. Met Hooks is het mogelijk om verschillende React-functies van je componenten te gebruiken. Er werd voornamelijk gebruik gemaakt van UseState en UseEffect hook. UseState laat een component informatie "onthouden" zoals bijvoorbeeld gebruikersinvoer. UseEffect stelt je in staat om code uit te voeren op bepaalde momenten tijdens de levenscyclus van een component, zoals na het renderen, bij het bijwerken van de state of bij het verwijderen van de component.


Componentstructurering en State Management: Het begrijpen van de belangrijkheid van goed gestructureerde componenten is ook een topic in de cursus. Er werd aandacht besteed aan best practices voor het organiseren en samenstellen van React-componenten. Goed gestructureerde componenten maken de schaalbaarheid en het onderhoud van de applicatie veel gemakkelijker. In React kan je gegevens tussen de verschillende componenten met behulp van props. In de cursus wordt er gebruik gemaakt van Redux voor global statemanagement. Er werd dieper ingegaan in de drie belangrijkste concepten van Redux namelijk actions, reducers en de store. Met Redux kan de globale applicatiestate beheert worden en de gegevensstroom efficiënt afgehandeld worden tussen meerdere componenten. Door useDispatch en useSelector hooks te gebruiken, kan de Redux-state succesvol geïntegreerd worden in de React-componenten.

Wat betreft de backend ontwikkeling wordt er in de cursus gebruik gemaakt van Express.js. er wordt uitgelegd hoe je onder andere Restful API’s kunt bouwen, custom error handling met middleware, JWT authenticatie en routing. Voor gegevens op te slaan wordt er gebruik gemaakt van MongoDb. Deze combinatie maakt het mogelijk om CRUD operaties uit te voeren. Deze concepten waren reeds gekend door de opleiding dus hier kon ik op zich sneller over gaan.



De code kan gevonden worden op Github onder de map Kidzplay: Link naar GitHub.


Reflectie

Aan de hand van de MERN-stack heb ik een e-commerce website gemaakt waar mensen speelgoed kunnen huren. Het idee is eigenlijk van mijn vriendin gekomen. Met zelf twee kleine kindjes in huis kan er nooit genoeg speelgoed zijn. Maar om te vermijden om telkens nieuw speelgoed te kopen dat na een tijdje ergens in een hoekje beland, is het idee gekomen om de mogelijkheid te bieden om speelgoed te huren. Zo kan je ook regelmatiger ‘nieuw’ speelgoed in huis halen en de kinderen hebben het niet door.

De cursus die ik heb gevolgd was boeiend en leerzaam. Het gebruik van de MERN-stack heeft mij laten zien dat het mogelijk is om op een relatief eenvoudige en snelle manier een full stack applicatie op te zetten. Met de backend t echnologie was ik al redelijk bekend met de concepten door mijn opleiding. Ik kon mijn opgedane kennis toepassen voor het werken met Express en het opzetten van de API's. Als oefening heb ikMongoDb in een Docker container laten lopen.

React.js is wel een nieuwe technologie voor mij. Omdat ik al kennis had van Angular,door de opleiding, begreep ik vrij de structuur en de opbouw van de componenten. Ik heb gemerkt dat er op dat vlak toch wel wat gelijkenissen zijn tussen Angular en React. Redux vond ik persoonlijk wel een uitdaging. Het verwerken en begrijpen van Redux kostte me iets meer tijd. Met de concepten actions, reducers en de store had ik niet direct door. Het kostte wat tijd en oefening om een goed begrip te krijgen van hoe Redux werkt en hoe ik het effectief kon gebruiken om de state van de applicatie te beheren.

Ik vond het een leuke uitdaging om een applicatie te maken volgens de MERN-stack. De cursus was goed opgebouwd en enorm leerrijk. Door het volgen van de cursus zijn mijn vaardigheden verbreed wat me alleen maar kan helpen naar de toekomst toe.
